### 攻防能量对比图

```
                    传统模式（你会输）

    攻击者                    防守者（你）
       ↓                         ↓
   "你太自私了"            →   "我不自私……"
       ↓                         ↓
   "你就是自私"            →   "我怎么自私了？"
       ↓                         ↓
   "又在狡辩"              →   （解释更多）
       ↓                         ↓
       → 无限循环，你精疲力竭 ←


                    反击模式（你会赢）

    攻击者                    反问者（你）
       ↓                         ↓
   "你太自私了"            →   "凭什么？"
       ↓                         ↓
   （愣住）                 →   "标准是什么？"
       ↓                         ↓
   （卡壳）                 →   攻守易势，胜负已分
       ↓                         ↓
       ← 对方陷入防守 ←
```

## 设计二：场景判断卡（红/黄/绿）

### 使用说明
遇到冲突时，快速判断属于哪个区域，选择应对策略。

---

### 红色区域（绝对不要吵）

```
┌─────────────────────────────────────────────────────────┐
│  🔴 红色区域：快速撤离                                  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  特征：                                                 │
│  • 对方情绪完全失控（尖叫、砸东西）                      │
│  • 对方有暴力倾向                                       │
│  • 对方是陌生人，且有攻击性                             │
│  • 对方喝醉了或神志不清                                 │
│                                                         │
│  策略：                                                 │
│  • 不眼神接触                                           │
│  • 不回话                                               │
│  • 快速离开现场                                         │
│  • 必要时报警                                           │
│                                                         │
│  核心原则：                                             │
│  不是认输，是战略止损                                   │
│  你的安全比什么都重要                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

### 黄色区域（谨慎处理）

```
┌─────────────────────────────────────────────────────────┐
│  🟡 黄色区域：谨慎处理                                  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  特征：                                                 │
│  • 对方是你需要维持关系的人（家人、同事、朋友）          │
│  • 冲突涉及实际利益（工作、金钱、权益）                  │
│  • 你状态一般，但不是最差                               │
│  • 问题可以讨论，但不是现在                             │
│                                                         │
│  策略：                                                 │
│  • 用技术反击，但不过度                                 │
│  • 反击后递梯子                                         │
│  • 约定稍后再聊                                         │
│  • 保持冷静，不升级冲突                                 │
│                                                         │
│  核心原则：                                             │
│  打但不打脸，给对方留台阶                               │
│ 看长期关系，不看单次输赢                                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

### 绿色区域（可以吵）

```
┌─────────────────────────────────────────────────────────┐
│  🟢 绿色区域：可以吵                                    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  特征：                                                 │
│  • 对方反复触碰你的底线                                 │
│  • 这次不反击，以后会被欺负得更惨                       │
│  • 你状态好，有精力应对                                 │
│  • 这是一个需要确立边界的时刻                           │
│                                                         │
│  策略：                                                 │
│  • 用全部技术反击                                       │
│  • 不要解释，直接反问                                   │
│  • 打疼对方，让他记住                                   │
│  • 确立边界，告诉对方什么是不可接受的                   │
│                                                         │
│  核心原则：                                             │
│  不是为了出气，是为了确立规则                           │
│  一次打疼，胜过百次忍让                                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 视觉元素使用原则

1. **少即是多**：每个页面不超过3种颜色、2种图标
2. **一致性**：相同类型的内容使用相同的视觉样式
3. **对比度**：确保文字与背景有足够的对比度（至少4.5:1）
4. **留白**：不要让页面过于拥挤，给读者呼吸的空间
5. **层次感**：通过字体大小、粗细、颜色区分信息层次
