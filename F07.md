## 四、快速入门作战卡（F01）使用说明

### 卡片内容

| 板块 | 内容 |
|------|------|
| 核心公式 | 攻击→停顿3秒→反问"凭什么？"→踢回去 |
| 三条铁律 | 不解释、不接球、不自证 |
| 万能起手式 | "凭什么？"及5种变体 |
| 适用场景 | 伴侣/父母/婆婆/同事/领导/亲戚/网友 |
| 不适用场景 | 垃圾人/醉汉/暴力倾向者 |
| 急救话术 | 3句直接用的话 |

### 使用场景

1. **贴书桌前**：遇到冲突时看一眼，想一想，再回应
2. **发给朋友**：帮助朋友应对家庭/职场冲突
3. **社群分享**：作为社群福利发放
