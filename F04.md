## 需修改案例及重写

### 婆媳案例1：生活方式干预

**原文问题**
- 反击语气过于平静温和，不符合真实婆媳冲突中的紧张感
- 话术太礼貌，缺少"被吵醒的烦躁"情绪铺垫

**重写版本**

```
【场景重现】
早上6:05，卧室门被用力敲响。
婆婆："太阳都晒屁股了还睡！年轻人就知道懒！"

【错误示范】
媳妇："妈，我昨晚加班到很晚……"
（解释！直接输掉）

【正确示范】
媳妇：（猛地拉开门，眼眶发红，声音沙哑但平静）
"妈，我昨晚项目上线，通宵没睡。你6点敲门，我刚睡着又被吵醒。
我不是懒，我是真的累。
你关心我，我谢谢。但这样敲门，我接下来一整天都没法工作。
你能不能……让我自己决定什么时候起床？实在不行，我搬出去住也行。"
```

### 婆媳案例2：育儿干预

**原文问题**
- 医生权威牌在婆媳场景中往往无效，反而激化矛盾
- 原理解释薄弱

**重写版本**

```
【场景重现】
婆婆偷偷给3岁的孙子喂巧克力。

【错误示范】
媳妇："妈，医生说他不能吃甜的！"
（婆婆内心os：医生懂什么，我养大几个了）

【正确示范】
媳妇：（蹲下来，温柔地擦掉孩子嘴角的巧克力）
"妈，我知道你是疼他。但昨天他牙龈发炎，医生说尽量别吃甜的。
你看，他这两天因为牙疼都没怎么吃饭。
下次你想给他好吃的，可以先问我一声吗？或者我买一些健康的小零食放家里？"

【原理解释】
为什么这样有效？因为你没有否定她的爱，而是给她一个新的表达爱的出口。
"疼孙子"和"遵守医嘱"不是对立的，你可以帮她找到一个两全的方式。
```

### 婆媳案例7：生活习惯差异

**原文问题**
- 分摊电费的解决方案不够"爽"
- 逻辑上对，但情感上缺少反击力度

**重写版本**

```
【场景重现】
婆婆："你天天洗澡，热水器开那么费电！我就舍不得天天洗！"

【错误示范】
媳妇："妈，我工作一天出汗了，不洗澡不舒服……"
（又变成解释）

【正确示范】
媳妇：（笑着点头）
"妈，你说得对，确实费电。
这样吧，从下个月开始，电费我来出。你就不用心疼了。
你节俭的习惯是好事，我们年轻人学不来。但你儿子赚钱，不差这点电费。
你就安心住着，别操这个心。"
```

## 总体评价

| 维度 | 平均分 | 评价 |
|------|--------|------|
| 真实感 | 7.2/10 | 较好，场景选择贴近生活 |
| 错误示范 | 7.4/10 | 较好，指出了常见错误 |
| 正确示范 | 7.3/10 | 较好，但部分案例偏书面化 |
| 原理解释 | 7/10 | 及格，部分案例解释不足 |

### 主要问题汇总

1. **语气过于平静**：部分正确示范缺少"被冒犯后的情绪"，读者难以产生代入感
2. **书面腔问题**：某些对话像是写出来的，不是说出来的
3. **原理解释可补充**：部分技术点可以更详细

### 建议优化方向

1. 增加情绪描写和场景氛围铺垫
2. 对话中增加停顿、省略号、语气词等口语元素
3. 每个案例后增加"为什么这样有效"的详细解析
