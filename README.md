# 《凭什么》

**一句话：四个字的反击公式——"凭什么？"**

一本教你在被攻击时如何不卑不亢地保护自己的书。

---

## 本书定位

- **目标读者**：25-40岁，在职场/家庭/网络中经常被指责却无力反驳的人
- **核心方法**：反击不自证——被攻击时不解释、不自证、把举证责任踢回去
- **独特价值**：心理学底层逻辑 + 场景化话术 + 可撕页作战卡

---

## 核心公式

```
攻击 → 停顿3秒 → 反问"凭什么？" → 踢回去
```

**三条铁律：**
1. 不解释
2. 不接球
3. 不自证
