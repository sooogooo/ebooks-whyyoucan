## 金句创作指南

### 创作公式

**公式1：颠覆认知**
```
[普遍相信的观点] + [但真相是相反的]
例：大家都觉得"解释清楚对方就会理解"，
但真相是"你一解释就输了"
```

**公式2：行动指令**
```
[遇到X情况] + [不要Y] + [而要Z]
例：遇到攻击时，不要解释，而要反问"凭什么？"
```

**公式3：比喻替换**
```
[抽象概念] + [用具体事物比喻]
例："沉默" → "敢于沉默的人才是真正有底气的人"
"反击" → "核武器最大的作用是躺在发射架上"
```

**公式4：数字强化**
```
[概念] + [具体数字]
例："三个筛选问题"
"一句话，四个字"
"10秒能让你崩溃10小时"
```

### 金句检查清单

写完金句后，检查是否符合：

- [ ] 可截图（单独看有冲击力）
- [ ] 可记忆（朗朗上口，容易复述）
- [ ] 有态度（观点鲜明，不是正确的废话）
- [ ] 符合调性（冷静、一针见血、不油腻）
- [ ] 避免鸡汤（不是"你要爱自己"式废话）
- [ ] 避免说教（不是"你应该怎样"）
